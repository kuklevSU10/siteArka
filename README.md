# АРКА — сайт студии

Статический сайт архитектурно-дизайнерской студии АРКА (vanilla HTML/CSS/JS).

- `index.html` — одностраничный лендинг
- `styles.css` — стили (токены, hero, секция-арка, секции)
- `script.js` — меню, reveal, превью экспертизы, форма
- `assets/` — изображения, self-hosted шрифты (Cormorant Garamond + Manrope), логотип, favicon

## Деплой
Публикуется на GitHub Pages автоматически при пуше в `main`
(workflow `.github/workflows/deploy.yml`). Включить один раз:
**Settings → Pages → Source: GitHub Actions.**

## История
Предыдущее Next.js-приложение сохранено в ветке `nextjs-archive` и теге `nextjs-snapshot`.
